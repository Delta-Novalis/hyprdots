These are dotfiles, files with a dot in the front, and they're used to configure the system.

Keybinds :
- Super + Enter = Terminal
- Super + R = Wofi
- Ctrl + Shift + Esc = Btop

- Super + Q = Kill
- Super + F = Fullscreen
- Super + V = Float
- Super + P = Pseudo
- Super + J = Split
- Alt + H = Lock

- Super + PrtSc = Region Screenshot
- Super + Shift + PrtSc = Monitor Screenshot
- Super + Ctrl + PrtSc = Window Screenshot

Installation :
```bash
git clone https://github.com/Delta-Novalis/hyprdots
chmod +x ~/hyprdots/scripts/install.sh
~/hyprdots/scripts/install.sh
```
